Quick Installation

Get up and running in minutes

1

Install Ollama

Download and install Ollama for local AI processing

curl -fsSL https://ollama.ai/install.sh | sh
2

Pull a Model

Download a language model for AI functionality

ollama pull llama3
3

Create Supabase Project

Create a Supabase project and get your credentials

# Visit app.supabase.com to create project
4

Install Logswise CLI

Install using Homebrew for easy setup and updates

brew tap k61b/tap brew install logswise-cli
5

🚀 One-Command Setup

Configure profile and automatically set up Supabase database

logswise-cli setup

New: Automatically creates database tables and schema!

6

⚙️ Initialize Database

Verify and complete Supabase database setup

logswise-cli init

Ensures: All database operations are complete and verified!

7

🧠 Enterprise AI Personalization

Configure advanced AI personalization for strategic intelligence

logswise-cli personalize

Unlocks: Professional context analysis, goal-driven development tracking, and adaptive communication styles!

Core Commands

Add a note to your collection

Add a note (short alias)
logswise-cli suggestion "query"
Get AI-powered suggestions based on your context
logswise-cli s "query"
Get suggestions (short alias)
logswise-cli chat "message"
Start an interactive chat with AI
logswise-cli c "message"
