Quick Installation

Get up and running in minutes

1

Install Ollama

Download and install Ollama for local AI processing

curl -fsSL https://ollama.ai/install.sh | sh
2

Pull a Model

Download a language model for AI functionality

ollama pull llama3
3

Setup Supabase

Create a Supabase project for note storage

# See SUPABASE_SETUP.md for detailed instructions
4

Install Logswise CLI

Install using Homebrew for easy setup and updates

brew tap k61b/tap brew install logswise-cli
5

Run Setup

Configure your profile and connections

logswise-cli setup

Command Reference

All the tools you need at your fingertips

Core Commands

logswise-cli note "text"
Add a note to your collection
logswise-cli n "text"
Add a note (short alias)
logswise-cli suggestion "query"
Get AI-powered suggestions based on your context
logswise-cli s "query"
Get suggestions (short alias)
logswise-cli chat "message"
Start an interactive chat with AI
logswise-cli c "message"
Chat with AI (short alias)

Productivity

logswise-cli interactive
Enter guided interactive mode for continuous workflows
logswise-cli recent --count 10
Display your most recent notes
logswise-cli completions bash
Generate shell completions for enhanced productivity
