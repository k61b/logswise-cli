How Logswise CLI Works
⚙️
1. Setup in Seconds
Run logswise-cli setup to create your profile and connect to your own Supabase & LLM. No cloud account needed.
📝
2. Take Notes Instantly
Add notes, todos, or code snippets from anywhere in your terminal. logswise note "Refactored login handler"
No context switching.
💡
3. Get AI Suggestions
Ask for context-aware tips, code reviews, or onboarding help. logswise suggestion "How to improve onboarding?"
🤖
4. Chat with Your LLM
Chat with your own local AI for help, brainstorming, or troubleshooting. logswise chat "Best way to log errors?"
🔒
5. Your Data, Always Private
All notes and profile data are stored locally. Only your notes are synced to your own Supabase. No personal data in the cloud.
