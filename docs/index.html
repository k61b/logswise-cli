📝Logswise CLI

Supercharge your terminal. Take instant notes, get AI-powered suggestions, and chat with your own local LLM. All private. All yours.
Get Started Free
⚡ Rust 🔒 100% Private 🤖 Local LLM 🗄️ Supabase Sync 💡 AI Suggestions 💚 Open Source 🖥️ macOS/Linux
🚀 See Logswise CLI in action:
Install in seconds:
$brew tap k61b/tap
$brew install logswise-cli
$logswise-cli setup
No account. No cloud. Works offline.
📝
Instant Notes
Log ideas, todos, and code snippets from your terminal. No context switching, no friction.
💡
AI Suggestions
Get actionable, context-aware tips powered by your notes, profile, and real context.
🤖
Local AI Chat
Chat with an AI assistant powered by your own LLM (Ollama). Get help, brainstorm, or ask anything—directly from your terminal.
🔒
Privacy-First
All notes and profile data are stored locally. Only your notes are synced to your own Supabase. No cloud lock-in, no vendor tracking.
